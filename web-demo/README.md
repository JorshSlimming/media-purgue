Web-demo for Media Purgue

- Dev: `npm install` then `npm run dev` inside `web-demo/`.
- Build: `npm run build` produces static files in `web-demo/dist` suitable for GitHub Pages.

Notes:
- This project reuses the React `App` from `apps/renderer/src` and installs a browser `window.mp` mock so the UI behaves like the desktop app.
- Run `npm install` in `web-demo` before using.
